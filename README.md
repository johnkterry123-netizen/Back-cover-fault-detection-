# 手机后壳故障检测系统

自动检测**0.1mm**以上的划痕，在**30件/分钟**的速度下，检测深色硅胶手机后壳的缺陷。

## 系统概述

该系统使用**图像处理**（非AI/ML）来检测微观缺陷：

```
传感器触发 → 图像捕获 → ROI提取 → 对比度增强
    ↓
与标���图比较 → 缺陷测量 → 尺寸检查 → 拒收/通过
```

---

## 工作原理（简单说明）

### 6步算法

**第1步：触发** 
- 激光断点传感器检测后壳到达
- 摄像机立即捕获图像

**第2步：找到零件**
- 将图像转换为黑白
- 找到最大的矩形（手机后壳）
- 裁剪其他部分

**第3步：修复光线**
- 深色硅胶上的深色划痕看不见
- 使用穹顶灯+低角度环灯创建白色阴影
- "拉伸"对比度：最暗的10%变成纯黑，最亮的10%变成纯白
- 现在划痕显示为明亮的白线

**第4步：寻找划痕**
两种方法：
- **方法A（推荐）：** 减去完美的"金色标准图像"
- **方法B（纹理表面）：** 模糊以去除晶粒，减去以隔离划痕

**第5步：测量尺寸**
- 计算缺陷中的像素
- 使用校准转换为毫米
- 如果长度>0.1mm则拒收

**第6步：拒收**
- 激活气喷嘴将零件吹下传送带
- 将缺陷记录到数据库

## 快速开始

### 1. 安装依赖项

```bash
pip install -r requirements.txt
```

### 2. 校准（关键）

运行系统前，必须校准系统：

```bash
python src/calibration_tool.py
```

该交互式工具将：
- **校准像素-毫米转换**（用于0.1mm精度）
- **创建黄金参考图像**（完美后壳）
- **测试图像处理设置**（阈值、模糊、对比度）

### 3. 运行检测

```bash
python src/main_detection.py
```

按 't' 模拟传感器触发（或修改为硬件触发）

## 故障排除

### 问题：未检测到划痕

**按顺序检查：**
1. **光线角度** - 必须使用斜光创建阴影
2. **黄金图像** - 真的是完美的吗？
3. **对比度设置** - 运行对比度测试工具
4. **阈值太高** - 降低 pixel_difference_threshold

### 问题：太多误检（灰尘被标记为缺陷）

**按顺序检查：**
1. **阈值太低** - 检测到所有东西
   - 解决：增加 pixel_difference_threshold
2. **镜头脏污** - 灰尘看起来像缺陷
   - 解决：用镜头布清洁摄像机
3. **光线不一致** - 反光看起来像缺陷
   - 解决：稳定光线强度
4. **金色图像陈旧** - 参考图有灰尘/标记
   - 解决：从完美零件创建新的金色图像
